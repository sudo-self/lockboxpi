Raspberry Pi SSD + SD Partition Layout

500GB SSD (sda)

sda1: 128MB — /boot/firmware (FAT32, boot)
sda2: 229GB — / (DietPi root)
sda3: ~250GB — /mnt/data (ext4, ready for future OS or storage)

SD Card (optional BerryBoot)

FAT32: BerryBoot files — boot menu / installer

SSD-first boot from sda2 (DietPi), SD card optional for multiboot/installer.
